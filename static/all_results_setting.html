Experiment Setting

  • ST-ResNet

    The best parameters found are following.

    xxxxxxxxxx
    args = {
      'dataset': 'DiDi',
      'city': 'Chengdu',
      'num_residual_unit': 4,
      'conv_filters': 64,
      'kernel_size': 3,
      'lr': 1e-5,
      'batch_size': 32
    }

    We can modify city parameter to Chengdu or Xian in ST_ResNet.py , and then run it.

    xxxxxxxxxx
     python ST_ResNet.py
